1. Activating the Print Screen Key#

The Print Screen key is a useful tool used for capturing screenshots and saving them as image files. It is available on most laptop keyboards, usually somewhere near the top right. In this guide, we will walk you through the process of enabling the Print Screen key on a Samsung laptop running Windows 8.

First, open the Charms Bar by pressing the Windows key + C, then select the Settings Charm. Once in the Settings Charm, click on the Change PC Settings option. In the PC Settings window, select the General tab and scroll down to the Keyboard section. Here, you will find a checkbox labelled Print Screen Key. Make sure this is ticked in order to enable the Print Screen key.

Next, press the Windows key + R on your keyboard to open the Run dialog box. Type in ‘screenshot’ and press enter. This will open the Snipping Tool. Whenever you press the Print Screen Key, the screenshot will be automatically stored in the Snipping Tool window.

Finally, press the Windows key + X on your keyboard to open the Power User Menu. From here, select the Disk Management option. Select the drive that you want to save your screenshots on and right-click it. Select the Properties option and then select the Security tab. Make sure that you have the necessary permissions to save the screenshots in this location.

With the Print Screen Key enabled and the necessary permissions set, you are now ready to take screenshots on a Samsung laptop running Windows 8.

Q2: What are the keyboard shortcuts for taking a Screenshot on Windows 8?

A: The keyboard shortcut for taking a screenshot on Windows 8 is the “Windows Key + PrtScn” combination. This combination will instantly copy the current screen display to the clipboard.

Q3: In which folder do Screenshots get saved?

A: On Windows 8, screenshots get saved in the Pictures folder, in a Screenshots sub-folder.

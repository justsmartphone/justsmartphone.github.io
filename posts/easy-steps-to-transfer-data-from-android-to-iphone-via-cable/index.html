3. Transferring the Data#

Transferring data from your Android to an iPhone may feel overwhelming, but with the right accessories, it’s the perfect task for the average user.   1. A USB Type C to Lightning Cable – this cable will allow you to connect your Android device to an iPhone using the phone’s ports.

  2. An OTG (On The Go) Adaptor – this adaptor allows the user to connect the USB Type C of the Android device to the USB port of their computer.

  3. The latest version of Android File Transfer App – this app can be used to transfer files from your Android device to the iPhone.

  4. A USB Type C to Type A connector – this connector allows the user to connect the USB Type C port of the Android device to the USB Type A port of the computer.
